# AURA | Premium Custom Business Website

A premium, modern, responsive multi-page digital agency website. Designed with luxury, minimal, and Apple-inspired aesthetic principles, complete with fluid motion transitions, light/dark modes, custom modal sliders, and micro-interaction analytics.

---

## 🎨 Visual Identity & Key Concepts

*   **Swiss Design Influence:** Elegant font hierarchies using *Plus Jakarta Sans* paired with structural display weights from *Space Grotesk*.
*   **Awwwards Quality Motion:** Fluid page swaps and staggered card entries using standard `motion/react` spring systems.
*   **Glassmorphic Accents:** Blurred frosted-glass navigation bars, glowing background ambient blobs, and sleek border boundaries.
*   **Complete Light & Dark Calibration:** Persisted user-mode selection with zero visual flickering on hydration.

---

## 🛠️ Tech Stack & Architecture

*   **Framework:** React 19 + TypeScript.
*   **Build Utility:** Vite 6.
*   **Aesthetic Styling:** Tailwind CSS v4 utility mapping.
*   **Motion Transitions:** `motion/react` (formerly framer-motion).
*   **Asset Vectors:** Lucide React icons.

---

## ⚙️ Core Interactivity Modules

1.  **State-Hydrated Router:** Instant, client-side route transitions featuring fluid fade and shift animations.
2.  **High-Fidelity Portfolio Showcase:** Categorized grid lists filtering projects with interactive case study slide sheets.
3.  **Active Service Accordion Tabulars:** Seamless capabilities tab maps linking directly to custom specs sheets.
4.  **Live Essay Search:** Interactive search and tag filtering system with zero latency.
5.  **Testimonial Slider Carousel:** Stateful transition boxes displaying vetted client quote histories.
6.  **Interactive FAQ Accordions:** Spring-loaded question panels with dynamic height allocation.
7.  **Briefing Configurations Forms:** Secure contact form and newsletter triggers with toast indicators.
8.  **Responsive System Maps:** Beautiful dark/light responsive vector map overlays pinpointing headquarters coordinates.
